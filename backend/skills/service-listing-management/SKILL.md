---
name: service-listing-management
description: Collect and submit provider listing details for groomers and dog walkers. Use when users ask to add, list, register, or publish a pet service listing.
---

## When to Use
- Use when the user wants to onboard as a provider or publish a service.

## Tools
- `add_service_listing`

## Workflow
- Extract as many provider fields as possible from the latest user message.
- Ask only for one missing required field at a time.
- Submit listing only after all required fields are complete.
